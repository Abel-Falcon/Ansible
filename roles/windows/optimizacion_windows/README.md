# Rol: optimizacion_windows

Automatiza tareas de optimización de rendimiento y desactivación de servicios innecesarios en Windows 10/11.

## Funcionalidades:
- Desactiva telemetría y servicios sin uso.
- Ajusta energía y red para máximo rendimiento.
- Limpia temporales.
- Desactiva efectos visuales.

## Variables clave:
- `opt_servicios_deshabilitar`: lista de servicios a desactivar.
- `opt_limpiar_temp`: limpia archivos temporales.
- `opt_modo_energia`: high performance / balanced.
- `opt_hibernacion_desactivar`: desactiva hibernación.
- `opt_desactivar_efectos`: desactiva animaciones.
- `opt_tune_network`: mejora parámetros TCP/IP.
